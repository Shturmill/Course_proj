# Course

Курсовой проект для ВУЗа 

СУБД для парковок автомобилей

стек:
* C#
* SQL/T-SQL

В данном проекте представлены скрипты для:

* [Создания](./DB/CREATE.sql)

* [Наполнения](./DB/insert.py)

* [Процедур](./DB/PROCEDURE.sql)

* [Тригеров](./DB/TRIGGER.sql)

* [Функций](./DB/FUNC.sql)

Данные скрипты для языка SQL отображают назначение и пользование вышеперечисленных объектов БД

## Overview

Данный проект представляет собой очень малую ценность в плане проги,просто обычная механическая работа на проверки (тем более на winforms), правда, времени угроблено немеренно, а так он беспонтовый пирожок.
